# Jogo Pedra, Papel e Tesoura (Pygame)

Jogo simples de **Pedra, Papel e Tesoura** feito em **Python** com **Pygame**.

## Requisitos

- Python 3.9+ instalado

Instale as dependências:

```bash
pip install -r requirements.txt
```

## Como jogar

No terminal, dentro da pasta do projeto:

```bash
python main.py
```

- Clique em **Pedra**, **Papel** ou **Tesoura** para jogar contra o computador.
- O placar é exibido na parte superior.
- Pressione **R** para limpar o resultado atual.
- Pressione **ESC** para sair do jogo.
